This utility allows you to render unicode-aided color, style and formatting the command line from your node.js scripts.

## Highlights
- No dependencies
- Ability to nest styles (Same like Chalk bcs its from Chalk library)
- Clean and focused
- 100% TypeScript
- Highly performant
